# Sistema de Gestión de Nuevo Ingreso - FIAD UABC

## 📝 Descripción
Herramienta desarrollada para los docentes de la Facultad de Ingeniería, Arquitectura y Diseño (FIAD) para automatizar la transición de alumnos desde su inscripción inicial hasta su ingreso formal.

## 🚀 Características Principales
- **Carga de Datos:** Integración con hojas de Excel.
- **Asignación Inteligente:** Algoritmo de distribución de grupos.
- **Gestión Multidisciplinaria:** Administración de grupos mixtos para Ingeniería y Arquitectura.
- **Seguimiento Académico:** Registro de asistencia y calificaciones de exámenes diagnósticos.

## 🛠️ Tecnologías
- **Lenguaje:** PHP
- **Base de Datos:** MySQL
- **Librerías:** PhpSpreadsheet para manejo de archivos Excel.
- **Frontend:** HTML5, CSS.
